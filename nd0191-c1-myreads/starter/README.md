# MyReads Project

MyReads project is a bookshelf app that allows you to select and categorize books you have read, are currently reading, or want to read. and you can also search for books by the author or tiltle of the book

## What to do to Run The Project

- install all project dependencies with `npm install` in your terminal
- start the development server with `npm start` in your terminal
- it will auto run on the port "3000"
